# Task Manager

This is a simple **Task Manager Web Application** built using **HTML, CSS, JavaScript, Node.js and Express.js**.  
It allows users to organize and manage daily tasks through an interactive and visually appealing interface.

The application supports creating, editing, completing, filtering and deleting tasks while maintaining a clean **glassmorphism UI design** with a modern background.

https://task-manager0715.netlify.app/

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks using **All / Active / Completed**
- Clear completed tasks
- Automatic task counter
- Timestamp for task creation
- Input validation for empty tasks
- Enter key support for faster task creation
- Clean glassmorphism UI design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## Purpose

This project was created as a small **full-stack development exercise** to practice:

- Building backend APIs using **Node.js and Express**
- Implementing **CRUD operations**
- Connecting frontend with backend services
- Designing a clean and interactive user interface
